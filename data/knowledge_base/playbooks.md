# Internal Playbooks — Demo Knowledge Base

## Gift distribution triage
- Verify employee roster synchronization.
- Check whether the employee is eligible in the client account.
- Retry the action after a refresh if the employee was added recently.
- Escalate only when the issue persists with identifiers and timestamps.

## Platform access triage
- Confirm user email address.
- Check whether an invite link expired or was consumed.
- Resend only when allowed by policy.

## Card / payment triage
- Validate merchant category and benefit scope.
- Do not assume every card decline is a platform bug.
- Escalate if the observed behavior contradicts the documented usage rules.
